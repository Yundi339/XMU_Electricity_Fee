# XMU_Electricity_Fee

这里是爬取厦大电费
=

修改xiaoqu,yuanqu,sushe这三个参数就能使用。<br>  
[厦大电费网址]http://elec.xmu.edu.cn/PdmlWebSetup/Pages/SMSMain.aspx <br>  
这个网站描述的信息花里胡哨，使用前最好先查看一下所要填的信息在网站里面的真实描述名称。 <br>  
代码内提供了获取校区，园区的相关代码，如果是非个人使用用途上，可以自由发挥。 <br>  
使用案例：<br><br>  
-
代码内修改：<br>  
    xiaoqu = '本部南光区' <br>  
    yuanqu = '南光7'<br>  
    sushe  = '0301'<br><br> 
运行输出：<br>  
    本部南光区 南光7 0301:    账户余额：28.81元    剩余电量：54.36度 <br>
